# SparkFun Analog/Digital MUX Breakout - CD74HC4067

This is a breakout board for the CD74HC4067 16-channel analog/digital multiplexer/demultiplexer.

Product Link: [https://www.sparkfun.com/sparkfun-analog-digital-mux-breakout-cd74hc4067.html](https://www.sparkfun.com/sparkfun-analog-digital-mux-breakout-cd74hc4067.html)

## Features
- 2V to 6V operation
- 16 channels can be multiplexed to one signal pin
- Bidirectional switches
- Enable pin to disable all switches
