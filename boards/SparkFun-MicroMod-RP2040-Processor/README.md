# SparkFun MicroMod RP2040 Processor

Recreation of the [SparkFun MicroMod RP2040 Processor](https://www.sparkfun.com/products/17720) in tscircuit.

## Features
- RP2040 dual-core ARM Cortex-M0+ microcontroller
- 128Mbit (16MB) external QSPI flash
- MicroMod M.2 edge connector interface
- Status LED
- 12MHz crystal oscillator
