# SparkFun Qwiic Shield for Thing Plus

The SparkFun Qwiic Shield for Thing Plus connects the I2C bus (GND, 3.3V, SDA, and SCL) from a Thing Plus/Feather form factor development board to four SparkFun Qwiic connectors.
