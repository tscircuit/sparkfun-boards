# SparkFun MicroMod STM32 Processor

SparkFun MicroMod processor board with STM32F405RGT6.

## Features
- ARM Cortex-M4 @ 168MHz
- 1MB flash, 192KB RAM  
- 128Mb (16MB) external SPI flash
- USB, SDIO, SPI, I2C, UART
- Standard MicroMod M.2 edge connector
