# SparkFun Tristimulus Color Sensor - OPT4048DTSR (Qwiic)

SparkFun Product: [SEN-22638](https://www.sparkfun.com/products/22638)

## Description

The OPT4048 is a high-speed, high-precision tristimulus XYZ color sensor from Texas Instruments. This Qwiic-enabled breakout features two Qwiic connectors for daisy-chaining and PTH pins for breadboard use.

## Features

- OPT4048DTSR XYZ color sensor (WSON-6)
- I2C interface (default address: 0x44)
- Supply voltage: 1.71V – 3.6V
- Two Qwiic JST-SH4 connectors
- 2.2kΩ I2C pullup resistors
- Board size: 1" × 1" (25.4mm × 25.4mm)

## Pinout (PTH Header J3)

| Pin | Signal | Description |
|-----|--------|-------------|
| 1   | VDD    | 3.3V power  |
| 2   | GND    | Ground      |
| 3   | SDA    | I2C data    |
| 4   | SCL    | I2C clock   |
| 5   | INT    | Interrupt   |
